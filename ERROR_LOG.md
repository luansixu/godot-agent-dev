## 格式规范

```markdown
### ERR-XXX: 错误简述

```

### ERR-001: 误用 MiniMax-Image 的图片理解功能
- **发现日期**: 2026-03-30
- **错误类型**: 认知
- **发生场景**: 尝试使用 `MiniMax-Image understand_image` 工具分析游戏截图
- **错误原因**: 误以为 MiniMax-Image MCP 有图片理解功能，实际只有图片生成
- **解决方案**: 确认有两个不同的 MCP - MiniMax MCP 有 `understand_image`，MiniMax-Image MCP 只有 `generate_image`
- **状态**: ✅ 已解决

2. **高频错误**: 某类型错误 > 5 个 → 创建专项规则
3. **已解决错误**: 解决后保留 30 天 → 移至 MEMORY_COLD.md
